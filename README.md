 Legal Document Analyzer

 📌 Project Description
This project analyzes legal documents and provides:
- Word count
- Agreement count
- Clause detection
- Penalty detection
- Summary generation
- Voice readout feature

 🚀 Features
- Upload legal documents
- Automatic analysis
- AI-based summary
- Text-to-speech (Read Summary)

🛠️ Technologies Used
- Java (Spring Boot)
- HTML, CSS, JavaScript
- REST API

▶️ How to Run
1. Run Spring Boot Application
2. Open browser → http://localhost:8080
3. Upload document and analyze
